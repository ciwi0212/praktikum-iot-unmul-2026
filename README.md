Project: Sistem Monitoring Kontrol berbasis IoT menggunakan ESP32 yang terhubung ke Telegram Bot.

Sistem ini memiliki fitur:

1. Kontrol LED melalui Telegram Bot
2. Monitoring suhu & kelembapan menggunakan sensor DHT22
3. Monitoring intensitas cahaya menggunakan sensor LDR
4. Notifikasi otomatis jika suhu/kelembapan melewati batas tertentu

Komponen yang digunakan:
1. ESP32
2. Sensor DHT22 (Suhu & Kelembapan)
3. Sensor LDR (Light Dependent Resistor) module
4. 1 LED
5. Kabel jumper
